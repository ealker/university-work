# APOD

Playing with NASA's APOD API - enter your birthday and retrieve the image and information about the APOD on that day.

> Yes I know hard coding an API key is naughty.

## Run Locally

1. Download folder.
2. Navigate to `index.html` on browser.
3. Enjoy.
